"Definitions of Managed Objects for IEEE 802.3 Medium Attachment Units (MAUs) using SMIv2", Keith McCloghrie, Donna McMaster, Dan Romascanu, S. Roberts, K. De Graaf, 11/05/1997. (93119 bytes)
This memo defines an experimental portion of the Management Information Base (MIB) for use with network management protocols in the Internet community. In particular, it defines objects for managing 10 and 100 Mb/second Medium Attachment Units (MAUs) based on IEEE Std 802.3 Section 30, ''10 & 100 Mb/s Management,'' October 26, 1995. This memo does not specify a standard for the Internet community.
