電腦網路程式作業二

目的:

    1. 熟悉socket程式介面
    2. 了解如何利用socket撰寫網路應用程式

內容:

    1. 寫一簡單的talk程式。基本要求是雙方可以以英文交談。
    2. 程式分兩個，一個是 talk server (talkd)，另一個是talk client (talkc)。
    3. 你可提供更具親和力的使用者介面。例如，可以使用中文、可以分割畫面、
      可以上下左右移動cursor等。

程式繳交方式:

    1. 程式於七月十日繳交，不可遲交。
    2. 繳交時，須配合助教進行測試。測試時間另由助教公佈。
    3. 繳交時尚須附一份報告，其內容包括:程式簡介、程式流程及其解說、程式特色、 所遭遇的問題、討論與心得分享等。最後必須附上完整的原始程式碼。

參考資料

    1. 本課程所提供之講義(http://exodus.cs.ccu.edu.tw)
    2. W. R. Stevens, "UNIX Network Programming," Prentice Hall, 1991.
