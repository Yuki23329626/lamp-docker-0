UDP 是提供不可靠傳輸的服務。因 IP 本身就是不可靠的傳
    輸，所以 UDP 很簡單地利用了 IP 來提供服務 。 UDP 主要是增
    加傳輸層 multiplex 到 IP 的功能。UDP 的封包格式如下：

          0　　 　 　15　 　 　　　31
          source port
          destination port
          message length
          checksum
          data

    　　其中 message length 包 括 header 和 data ， 以 byte 為 單位。
    checksum 則和 TCP 一樣利用 pseudo header + header + data 進 行
    16-bit checksum。但因 UDP 是提供不可靠傳輸，所以也可以不做
    checksum，只要將 checksum 設成 0 即可。
