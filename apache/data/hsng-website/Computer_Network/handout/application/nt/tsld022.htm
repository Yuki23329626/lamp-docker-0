Gopher 伺服器

  • Gopher資料庫
    • 使用Tag file來描述檔案內容和型別
      gdsset -c -gn -f “檔案內容描述” -a <管理者名稱>
      -e <管理者email> filename
      • -g是指定檔案類型: 0:text, 1:dir, 5:DOS bin, 6:uuencoded, 7:WAIS, 8:Telnet term, 9:bin, c:calendar, g:gif, h:html, p:pdf, T:TN3270 term
    • 查Tag file: gdsset -r filename
    • 與其他網站連線
      gdsset -l -gl -f “網站名稱” -h連結位址 -pPort 目錄名稱
        gdsset -l -gl -f “中正Gopher” -hgopher.ccu.edu.tw -p70 ccu
