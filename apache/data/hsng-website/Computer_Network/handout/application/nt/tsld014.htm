NT的Name Resolution

  • WINS

    • 提供處理NetBIOS over TCP/IP主機名稱
    • 集中式的名註冊處理
      • Client開機時必須先向WINS伺服器註冊
      • 一網域只可以有一主WINS伺服器，但可以有多台次要WINS伺服器
    • 安裝與設定
      • 由控制台的網路視窗以新增網路服務安裝
      • 可設定更新、廢棄、廢棄等候、及驗證等時間
      • 可設定推入與協力電腦，互相交流資料庫(非階層式)
      • 對於非WINS Client機器，可以做靜態設定
