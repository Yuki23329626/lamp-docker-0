FTP 伺服器

  • 管理與設定

    • 記錄頁次可設定連線記錄頻率和記錄檔
    • 進階頁次可設定只允許或排除特定IP主機或IP子網路連線
    • 目錄說明檔
      • 可為每一目錄寫一說明檔 (annotation file)
      • 用Registry.exe在HKEY_LOCAL_MACHINE\ SYSTEM\CurrentControlSet\Services\MSFTPSVC\Parameters中新增AnnotateDirectories及值0x01
      • 在每一目錄下設定~ftpsvc~.ckm檔
