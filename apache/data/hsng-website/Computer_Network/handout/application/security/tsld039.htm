Digital Signature

  • Requirements

    • The receiver can verify the claimed identity of the sender. (Authentication)
    • The sender cannot later repudiate the contents of the message. (不可否認)
    • The receiver cannot possibly have concocted the message itself. (完整性，不可修改)
