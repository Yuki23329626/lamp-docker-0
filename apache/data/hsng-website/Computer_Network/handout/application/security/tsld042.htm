Authentication without Encryption

  • Some times we only want authentication, but do not care about message encryption

    • e.g., Message broadcast from authorized source
    • 轉帳超過100萬必須記錄(公開)

  • Solution: message digest (MD)

    • Use a secure (one-way) hash function H to compute a fixed-size tag H(M||SAB), called a message digest for a given message M concatenated with a shared secret value SAB between two communicating parties A and B.

  • MD 用在secret-key digital signature上

    • 不用傳KBB(A,t,P) ，只需傳KBB(A,t,MD(P))
