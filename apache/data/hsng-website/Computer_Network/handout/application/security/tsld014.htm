DES: Data Encryption Standard

  • Details of each iteration

    • 將64-bit input 分成兩個32-bit (Li-1,Ri-1)
    • 將Ri-1 expand 成48-bit (E) (Transposition + Duplication)
    • 將E與Ki (48-bit key) XOR
    • 將上面的結果分成8個6-bit blocks
    • 將6-bit block input到 S-BOX，並map成4-bit
    • 將8個4-bit blocks input 到 P-BOX
