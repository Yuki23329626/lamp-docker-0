Run DES Couple Times

  • Breaking is as hard as running DES once

    • If we know (Pi, Ci) for some i
    • Ci = Ek2 ( Ek1 (Pi) ) )
    • Build a Ra = Ea (P1) table (a is the key, table size is 256)
    • Build a Sb = Db (C1) table (b is the key, table size is 256)
    • Find key pairs such that Ra = Sb
    • Use these key pairs to try other (Pi, Ci) pairs
