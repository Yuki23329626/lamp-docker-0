Encryption Methods

  • One-time pad

    • Choose a random bit string
    • XOR with the binary form of the plaintext
    • Symmetric key
    • Needs a long bit string if plain text is long
    • Could store plenty of keys on a CD-ROM
      • Use a number to indicate which key is used
    • Almost unbreakable
