Authentication Using Kerberos

  • Developed by MIT, based on a variant of Needham-Schroeder Protocol.

  • Three servers: AS: Authentication Server, TGS: Ticket-Granting Server, 對方(Bob) server

  • Alice need only a password to securely communicate with Bob and authenticate the communication.

  • To support multiple realms, each with its own AS and TGS.

    t: a timestamp
