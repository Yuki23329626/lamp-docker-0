Encryption Methods

  • Substitution cipher

    • Each letter or group of letters is replaced by another letter or group of letters
    • Key is pairing between plaintext characters and ciphertext characters
    • Symmetric key: sender/receiver hold the same key
    • 26! (約1026) possible keys
    • Vulnerable to decryption using observed frequency of letters
      • For example, “e” and “the” appear very often
