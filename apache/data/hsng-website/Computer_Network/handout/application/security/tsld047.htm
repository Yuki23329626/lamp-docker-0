MD5 Algorithm

  • The 128-bit buffer is represented as four 32-bit registers(A,B,C,D) which initialized to the following hexadecimal values:

    • A = 01234567
    • B = 89ABCDEF
    • C = FEDCBA98
    • D = 76543210

  • The table T[1…64] constructed from the sine function where T[i] = integer part of , i in radians. This table provided a “randomized” set of 32-bit patterns for eliminating regularities in the input data.
