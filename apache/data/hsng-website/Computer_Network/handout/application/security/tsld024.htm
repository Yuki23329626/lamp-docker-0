RSA Algorithm

  • The algorithm (discovered by Rivest, Shamir, and Adleman at MIT in 1978)

    • 1. Choose two large primes, p and q . (> 10100)
    • 2. Compute n=p*q and z=(p-1)*(q-1) .
    • 3. Choose a number relatively prime to z and call it d .
    • 4. Find e such that e*d=1 mod z .

  • For plaintext P, 0 £ P £ n (664-bit block is n ~ 10200)

  • encryption key = (n,e) decryption key = (n,d)

  • The security of the method is based on the difficulty of factoring large numbers.
