Secure Hash Functions

  • Properties of a secure hash function H:

    • H can be applied to a block of data of any size.
    • H produces a fixed-length output.
    • H(x) is relatively easy to compute.
    • For any given code m, it is computationally infeasible to find x such that H(x) = m.
    • For any given block x , it is computationally infeasible to find y 1 x with H(y) = H(x).
    • It is computationally infeasible to find any pair(x,y) such that H(x) = H(y). For protecting against the birthday attacks.
