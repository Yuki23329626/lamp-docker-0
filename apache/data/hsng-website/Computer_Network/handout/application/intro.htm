目前網路應用軟體很多，每一種網路工具均有其使用的協定。在此，我們介紹多種
協定:DNS是將hostname轉成IP address的協定，SMTP(含MIME)是送電子郵件的協定，
PGP和PEM是安全電子郵件的協定，POP3是Mail Agent使用的協定。HTTP是WWW所使用的
協定，HTML是WWW文件的描述語言，Java Script是可放在WWW文件中的程式語言，Java
Applet提供WWW文件中的互動、動畫效果等。在CGI語言部份，我們介紹一下Perl語言。
在網路管理方面，我們介紹SNMP, ASN.1 和 BER運作原理，資訊安全則介紹資訊加密技
術、認證和數位簽章。網路安全我們探討網路駭客行為和防火牆技術。最後我們介紹目
前Internet上的電子商務所需技術和NT上相關的網路工具。
