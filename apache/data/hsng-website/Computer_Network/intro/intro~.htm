課程目標

    • 熟悉電腦網路基本概念
    • 熟悉網際網路的發展 (protocols, applications, ...)
    • 學會網路程式設計(Socket, Java)
    • 體驗網路學習環境的好處 (Asynchronous Learning Environment)

    參考書

授課大綱

  • Data Transmission (Layer 1)
    • Transmission media
    • Local asynchronous communication (RS-232)
    • Long-distance communication (Modem)
  • Packet Transmission (Layer 2)
    • Framing, addressing, error detection
    • LAN technology
    • LAN interconnecting
    • LAN and WAN
  • Internetworking (Layer 3, 4)
    • Routing Protocols:IP, ICMP,ARP, IPv6
    • Reliable connections: TCP
  • Network Application (Layer 6,7)
    • Client-server model
    • Socket interface
    • DNS, Email, FTP, BBS
    • WWW
      • CGI
      • Java
    • Network management and security
