課程目標

    • 熟悉電腦網路基本概念
    • 熟悉網際網路的發展 (protocols, applications, ...)
    • 學會網路程式設計(Socket, Java)
    • 體驗網路學習環境的好處 (Asynchronous Learning Environment)
評分標準

    • 期中考: 35%
    • 期末考: 35%
    • 線上測驗: 0%
    • 作業(3次): 15%
    • 程式(2次): 10%
    • 網路學習環境參與度: 5%
教材

    教科書

    • A.S. Tanenbaum, Computer Networks, Prentice-Hall, 1996

    參考書

    • 黃能富, 區域網路與高速網路, 維科出版, 1996
    • D. E. Comer, Computer Networks and Internets, Prentice Hall, 1997.
    • W. R. Stevens, UNIX Network Programming, Prentice-Hall, 1991
    • D. Flanagan, Java in a Nutshell, O'Reilly&Associates, Inc., 1996
授課大綱

  1. Introduction
    1. Terminologies
    2. Layered network architecture
    3. Internetworking
    4. History
    5. Down load slides (Postscript)
  2. Physical Layer
    1. Theoretic basis
    2. Transmission media
    3. ISDN and Broadband ISDN
    4. Data transmission
    5. Data encoding/decoding
    6. Down load slides (Postscript)
  3. Data Link Layer
    1. Framing
    2. Error Control
    3. Flow Control
    4. HDLC Layer
    5. Down load slides (Postscript)
  4. MAC Layer
    1. Multiple access problem (part I)
      1. case study: IEEE 802.3, Fast Ethernet, Gigabit Ehternet, Ether Switch
      2. Down load slides (Postscript)
    2. Multiple access problem (part II)
      1. case study: IEEE 802.5, FDDI, DQDB, 100VG-anyLAN, Wireless LAN
      2. Down load slides (Postscript)
    3. Bridge
      1. Transparent bridge
      2. Source routing bridge
      3. Remote bridge
      4. Down load slides (Postscript)
  5. Network Layer
    1. Routing (part I)
      1. Link State
      2. Distance Vector
      3. ES-IS Routing
      4. Intradomain IS-IS Routing
      5. Interdomain IS-IS Routing
      6. case studies: IDRP, EGP, BGP, IP, ICMP, SLIP, PPP
      7. Down load slides (Postscript)
    2. Multicast and IP (Part II)
      1. Multicast (MBone)
      2. Congestion Control
      3. Flow Control
      4. IP, CIDR, IPv6
      5. Down load slides (Postscript)
    3. ATM and AAL
        4. Down load slides (PowerPoint)
  6. Transport Layer
    1. Reliable/Unreliable data transfer
    2. Connection setup/tear down
    3. Congestion and flow control
    4. Multiplexing and addressing
    5. case studies: TCP, UDP
    6. QOS Signaling Protocol: RSVP
    7. Socket programming
    8. Advanced socket programming
    9. Winsock programming
    10. IP Multicast Programming
    11. Remote Procedure Call
  7. Application Layer
    1. Network Security
    2. Firewall
    3. DNS
    4. EMail
      1. SMTP
      2. MIME
      3. POP3
      4. PGP and PEM
    5. WWW (HTTP)
      1. HTML
      2. Javascript
      3. Java
      4. Perl
    6. USENET NEWS
    7. Network Management (SNMP)
    8. E-Commerce
  8. Summary and Future Directions
